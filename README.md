# 0. 대회 개요
- 목적: 한국거래소의 정보사업을 확대하고, 시장참가자들에게 더욱 유용한 투자 참고 정보를 제공하기 위해, 경진대회를 개최. 'AI/머신러닝을 적용한 정보상품∙컨텐츠 개발' 등 금융 빅데이터 활용한 다양한 비즈니스 아이디어를 제시
- 주최: 한국거래소
- 참가대상: 개인(학생•직장인 등) 및 법인(스타트업 등)
